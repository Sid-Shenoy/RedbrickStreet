1) Zombie spawning (`spawnZombies.ts`)
1.1) Intent
1.1.1) Spawn a fixed number of zombies when the game begins (after the intro starts).

1.2) Constants (hard)
1.2.1) `ZOMBIE_COUNT` is a constant and defaults to 50.
1.2.2) `MIN_ZOMBIE_SPAWN_DIST_M` is a constant and defaults to 30.

1.3) Spawn constraints (hard)
1.3.1) Zombies may only spawn in first-floor rooms:
1.3.1.1) Use `house.model.firstFloor.regions`.
1.3.1.2) Do NOT use second-floor regions.
1.3.2) Exclude the first-floor `stairs` region (zombies cannot climb stairs).
1.3.3) No zombie may spawn within `MIN_ZOMBIE_SPAWN_DIST_M` meters (XZ distance) of the player at game start.
1.3.4) Collisions must be disabled for all zombie meshes:
1.3.4.1) `checkCollisions = false`
1.3.4.2) `isPickable = false`

1.4) Placement rules
1.4.1) Spawn positions are chosen randomly from first-floor regions, weighted by region area.
1.4.2) Spawn height is `FIRST_FLOOR_Y`.
1.4.3) Zombie model path is `/assets/models/zombie.glb`.
